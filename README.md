# ระบบติดตามงานค้าง (Student Missing Work Tracker)

แอพพลิเคชั่นสำหรับติดตามงานค้างของนักเรียน ใช้งานง่ายผ่านเว็บเบราว์เซอร์ รองรับการแสดงผลบนมือถือ

## วิธีใช้งาน
1. เปิดไฟล์ `index.html`
2. เพิ่มรายชื่อนักเรียน
3. ติ๊กเลือกงานที่ค้าง
4. ข้อมูลจะถูกบันทึกอัตโนมัติในเครื่อง (localStorage)

## การนำขึ้นเว็บ (GitHub Pages)
1. สร้าง Repository ใน GitHub
2. Push ไฟล์ขึ้นไป
3. เปิดใช้งาน GitHub Pages ใน Settings
